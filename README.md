# ✍️ Grammar Fixer & AI Detector Web App

A Hugging Face Space built with Gradio and powerful LLMs that offers:

- ✔️ Grammar correction using Phi-4 Mini
- 🎯 Tone improvement
- 🔄 Fluency enhancement
- 🌀 Paraphrasing using T5-base
- 🕵️ AI vs Human text detection (`roberta-base-openai-detector`)
- 📂 File upload (.txt)
- 💾 Output download
- 🔄 Unlimited input via chunking
